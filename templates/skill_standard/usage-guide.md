# Usage Guide: `skill-name-kebab-case`

## Installation

```bash
pip install -r requirements.txt
```

## Running the example

```bash
python examples/minimal_example.py
```

## Running tests

```bash
pytest tests/
```

## Integration with an Agent

Tell the agent: "Use the `skill-name-kebab-case` skill to ..."

The agent will read `SKILL.md` and generate code. Verify the generated code uses the recommended parameters.
